This application has been constructed with the various technologies I am currently using and studying. Some I have extensive knowledge in, while others I have only a working knowledge of.

Some of the technologies used in the construction of this application are listed below.

Let's start with the elephant in the room.

These are the core technologies used in creating this application. This is known as MEAN stack developement. The wonderful thing about the MEAN stack is that it is all JavaScript.

MongoDB

is a NoSQL database. MongoDB's great manual is the place to get started with NoSQL and MongoDB.

Express

Express is a NodeJS server-side application web framework. Check out The ExpressJS API reference for more information or StackOverflow for more info.

AngularJS

AngularJS is client-side web application framework. The Thinkster Popular Guide and Egghead Videos are great resources.

Node.js

Node.js is a JavaScript run-time, popular for being a web server platform. Node's website and this stackOverflow thread offer excellent starting points to get to grasps with node.
